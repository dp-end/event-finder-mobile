# 📱 EventFinder Mobile

## ✨ Öne Çıkan Özellikler
- **🔍 Gelişmiş Filtreleme:** Kategori ve tarihe göre etkinlik arama.
- **🎫 Bilet Yönetimi:** Uygulama üzerinden bilet görüntüleme ve rezervasyon.
- **🔐 Güvenli Giriş:** JWT tabanlı kimlik doğrulama sistemi.
- **🌓 Modern UI:** Temiz ve akıcı kullanıcı deneyimi.

## 🛠️ Teknoloji Yığını & Mimari
- **Framework:** [Flutter](https://flutter.dev/)
- **Dil:** Dart
- **State Management:** (Buraya kullandığın yapıyı yaz: Provider / Bloc / GetX)
- **API:** .NET Core Web API (RESTful)
- **Mimari:** Clean Architecture / MVVM (Hangisini kullandıysan)
